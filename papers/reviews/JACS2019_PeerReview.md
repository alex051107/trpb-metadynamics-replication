## 1. 方法学评估

### 增强采样方法
- **选择**：Well-tempered MetaDynamics + path collective variables
- **评价**：合理。Path CV (s, z) 能捕捉 COMM domain 的 O→PC→C 过渡，比简单的 RMSD-based CV 更适合描述多步构象变化。
- **潜在问题**：path CV 依赖参考路径的选择。用 1WDW (open) 和 3CEP (closed) 作为端点，但 3CEP 来自 StTrpS（不是 PfTrpB），这是一个跨物种参考。作者假设 closed state 几何学在物种间保守，但没有显式验证。

### 力场与采样充分性
- **力场**：ff14SB + TIP3P — 标准选择，适合蛋白质构象研究
- **温度**：350 K — 对应 P. furiosus 的生理温度，合理
- **采样**：10 walkers × 50-100 ns = 500-1000 ns per system — 对 path CV MetaD 来说属于中等偏上
- **潜在问题**：收敛性只通过 Figure S4/S5（能量差随时间变化）展示，没有给出 block analysis 或 error bar。不清楚最终 FEL 的误差有多大。

### PLP 参数化
- **方法**：GAFF + RESP (HF/6-31G(d))
- **评价**：标准做法。但 PLP 是共价结合到 K82 的，GAFF 对共价辅酶的描述可能不够准确。SI 没有提供实际的 mol2/frcmod 文件，无法验证。
- **风险等级**：中。如果 PLP 参数不准确，可能影响 K82-Q₂ 距离的计算——这恰好是 productive closure 的关键判据之一。

## 3. 结论强度评估

### 强结论（数据支持充分）
- ✅ PfTrpB0B2 比孤立 PfTrpB 有更低的 O→C 能垒（~2 vs ~6 kcal/mol）
- ✅ 孤立 PfTrpB 的 closed state 是"非生产性"的（K82-Q₂ = 3.9 Å > 3.6 Å 阈值）
- ✅ SPM 方法能识别 5/6 个 DE 突变位点

### 需要谨慎解读的结论
- ⚠️ "0B2 恢复了与 TrpS complex 相似的构象景观" — 相似但不完全一致，能垒值和 basin 位置有差异
- ⚠️ Productive closure 的 3.6 Å 阈值 — 这个数字是从 PfTrpS reference 得来的，不是独立确定的
- ⚠️ SPM 的预测能力 — 只在一个体系上验证（PfTrpS → 0B2），样本量太小

### 未验证的隐含假设
- ❓ Path CV 的两个端点（1WDW, 3CEP）能充分描述所有功能相关的构象变化
- ❓ 所有 4 个催化中间体共享相同的 path CV 定义是合理的
- ❓ 350 K 下的 FEL 定性特征在 room temperature 下也成立

## 4. 对我项目的影响

### 复刻时需要注意的坑
1. **PLP 参数化是最大风险**：SI 没给文件，必须自己做。做出来的参数和原文是否一致无法验证。
2. **3CEP 作为 closed endpoint**：这是 StTrpS 的结构，不是 PfTrpB。在 PfTrpB 体系上用这个端点做 path CV，可能引入偏差。
3. **收敛标准不明确**：原文没说"跑多久算够"。需要自己定义收敛判据。
4. **只有一次 run per condition**：即使完美复刻了 protocol，得到的具体数字可能因随机种子不同而有 ±1 kcal/mol 变化。

### 复刻成功的判据
基于以上分析，建议将成功标准放宽为：
- O→C 能垒趋势正确（0B2 < PfTrpB）
- 定量值在 ±1.5 kcal/mol 以内（而非 manifest 里写的 ±0.5）
- Productive closure 定性再现（0B2 能到 productive closed，PfTrpB 不能）
