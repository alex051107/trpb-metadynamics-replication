# TrpB MetaD — path 15 frames + MD parameter files

## path_15_frames/

PATHMSD 参考路径，15 个构象沿 1WDW(open) → 3CEP(closed) 线性插值。
112 Cα = COMM 97-184 + base 282-305（1WDW 编号）。

| File | What |
|---|---|
| `path_seqaligned.pdb` | 15-MODEL 多模型 PDB，atom serial 1..112 |
| `path_seqaligned_gromacs.pdb` | 同上但保留 GROMACS 全系统的 atom serial（production plumed.dat 用这个） |
| `frames/frame_01.pdb` … `frame_15.pdb` | 拆开的 15 个独立 PDB（每帧一个） |
| `endpoint_pdbs/1WDW.pdb` | open 端点（PfTrpB chain B） |
| `endpoint_pdbs/3CEP.pdb` | closed 端点（StTrpB chain B，offset +5 残基编号） |
| `build_seqaligned_path.py` | 重建脚本：NW 序列对齐 → Kabsch → 线性插值 |
| `verify_and_materialize_seqaligned_path.py` | Codex 独立复算（7/8 PASS vs Claude 版） |
| `split_frames.py` | 把 multi-MODEL 拆成 15 个单独 PDB |
| `summary.txt` | 几何摘要：⟨MSD_adj⟩=0.0228 Å², λ=100.79 Å⁻² |
| `VERIFICATION_REPORT.md` | Codex 验证报告 |
| `REVERSE_SANITY_CHECK.md` | 反向 sanity check |

**作者推荐方法**（JACS 2019 SI p. S3 原文）："A path of 15 conformations was
generated by linear interpolation between the X-ray available data."
2026-04-23 收到 Miguel 邮件给了 plumed.dat（LAMBDA=80），但**没**给 PATH.pdb
原文件。

**我们的方法 vs 作者**：方法论一致（linear interpolation），唯一工程差异是
跨物种残基编号必须先 Needleman-Wunsch 对齐（1WDW resid X ↔ 3CEP resid X+5）。
我们最初版本没做这一步，得到 λ=3.80 Å⁻²（错 21×）。修好后 λ=100.79 Å⁻²，
作者邮件 LAMBDA=80 Å⁻²，比值 **1.26×**（Branduardi 经验 tolerance ±2× 内）。

## md_parameters/

跑 MetaD 用到的所有 GROMACS / PLUMED / AMBER 输入文件。Tier 1.5 chain：
EM → NVT 1 ns → 5 个 ramp 阶段（HEIGHT 0.03→0.06→0.10→0.13→0.15 kcal/mol，每段 2 ns）→ prod 40 ns。

### topology/
| File | What | Source |
|---|---|---|
| `pftrps_ain.parm7` | AMBER topology（39,268 atoms，ff14SB+TIP3P+GAFF/PLP） | `tleap` 自动生成 |
| `pftrps_ain.inpcrd` | AMBER input coordinates | 同上 |
| `pftrps_ain_leap.pdb` | tleap 输出的 PDB（含水/离子，便于 PyMOL 可视化） | 同上 |
| `Ain_gaff.lib` | PLP cofactor library（GAFF + RESP charges） | 我们自己 antechamber/RESP 生成 |
| `tleap_pftrps_ain.log` | tleap 的完整 log（记录所有 leap 命令 + warnings） | tleap |
| `topol.top` | GROMACS topology | ParmEd 从 `pftrps_ain.parm7` 转出 |
| `conf.gro` | GROMACS coordinates | ParmEd 从 `pftrps_ain.inpcrd` 转出 |
| `convert_amber_to_gromacs.py` | ParmEd 转换脚本 | — |
| `ca_indices.txt` | 112 个 Cα 在系统里的 atom index（plumed PATHMSD 用） | — |

### mdp/
GROMACS run-control 参数（dt, nsteps, lincs, T-coupling 等）。
| File | Stage | nsteps × dt | Notes |
|---|---|---|---|
| `em.mdp` | energy minimize | 1000 steps steepest-descent | gmx 默认 |
| `nvt.mdp` | NVT 1 ns | 500,000 × 2 fs | gen_vel=yes, PLUMED off |
| `metad_stage1.mdp` … `metad_stage5.mdp` | 5 阶段 ramp，每段 2 ns | 1,000,000 × 2 fs | `lincs_iter=2`, continuation=yes |
| `metad_prod.mdp` | production 40 ns | 20,000,000 × 2 fs | 同上 |

### plumed/
| File | What |
|---|---|
| `plumed_stage1.dat` | HEIGHT=0.03，**无 RESTART**（首段，FP-037 修过） |
| `plumed_stage2.dat` … `plumed_stage5.dat` | HEIGHT 渐增到 0.15，**有 RESTART** |
| `plumed_prod.dat` | HEIGHT=0.15，有 RESTART |

所有 plumed 都用 `PATHMSD REFERENCE=path_seqaligned_gromacs.pdb LAMBDA=100.79`，
`METAD ARG=p1.sss,p1.zzz ADAPTIVE=DIFF SIGMA=1000 BIASFACTOR=10 PACE=1000 TEMP=350`，
`UPPER_WALLS ARG=p1.zzz AT=2.5 KAPPA=800`，`WALKERS_DIR=../HILLS_DIR_tier1p5`。
