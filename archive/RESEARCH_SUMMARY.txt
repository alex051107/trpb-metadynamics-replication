2. METADYNAMICS PARAMETERS DETERMINED
================================================================================

PATH COLLECTIVE VARIABLE (Path-CV):
- Connects open and closed reference structures
- Typical: 20-50 interpolated intermediate structures via g_morph
- σ variable: progress along path (0 = open, 1 = closed)
- λ/z variable: perpendicular distance penalty

WELL-TEMPERED METADYNAMICS:
- HEIGHT: 0.4-0.6 kJ/mol (~0.2 × k_B*T at 300 K)
- SIGMA: 0.05-0.15 for path; 0.1-0.3 nm for distances
- PACE: 500 MD steps between Gaussian deposits
- BIAS_FACTOR: 10-15 (controls convergence to free energy)
- GRID_SPACING: 0.05 (fine FES discretization)
- TEMPERATURE: 300 K (standard; check SI for exact value)
- TIMESTEP: 2 fs (standard for all-atom MD)

EXPECTED OUTCOMES:
- ΔG_conformational: 2-10 kcal/mol (domain closure cost)
- Barrier height: 5-15 kcal/mol
- Exchange timescale: microseconds to milliseconds

================================================================================
5. PRACTICAL SETUP STEPS
================================================================================

STAGE 1: Structure Preparation
- Download PDB 5DVZ and 5DW0
- Protonate and add hydrogens (pdb4amber)
- Generate intermediate structures via g_morph (30 frames)
- Align all structures to common reference

STAGE 2: AMBER Topology Building
- Parameterize PLP cofactor (GAFF + RESP)
- Build topology in LEaP using ff14SB
- Solvate with TIP3P (~8000 waters)
- Neutralize with Na+ ions
- Minimize & equilibrate (1000 steps min + 10 ps NVT)

STAGE 3: PLUMED Path-CV Definition
- Write plumed.dat with PATH directive linking morphed structures
- Define Gaussian deposition (METAD) with parameters above
- Validate path definition with 1 ns test run

STAGE 4: Test MetaDynamics Run
- 10 ps test simulation to validate setup
- Check HILLS file generation
- Verify CV exploration of full [0,1] range

STAGE 5: Production Run
- 100-500 ns per replica (depends on convergence)
- 4-8 parallel replicas recommended
- GPU acceleration (pmemd.cuda) strongly recommended

================================================================================
7. EXPECTED FES FEATURES
================================================================================

Free energy surface should show:
- Minimum at σ ≈ 0 (open state) → ΔG ≈ 0
- Minimum at σ ≈ 1 (closed state) → ΔG ≈ 2-10 kcal/mol
- Maximum at σ ≈ 0.5 (transition state) → ΔG‡ ≈ 5-15 kcal/mol
- Low perpendicular penalty (z ≈ 0-0.3)

Population distributions from evolved variants should show:
- Wild-type: favors open state
- Evolved: shift toward closed state stabilization
- Intermediate: populations increase with activating mutations
