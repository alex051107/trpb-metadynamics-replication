# .docx Formatting Spec

## Color palette

Black and white only. No colored backgrounds, no colored borders.

- All text: black (#000000)
- Table borders: black, thin (1pt)
- Table header row: bold text, no background color
- Page background: white

## Typography

- Font: Arial throughout
- Body text: 11pt
- Section headings: 14pt, bold
- Subsection headings: 12pt, bold
- Table text: 10pt
- Header table: 11pt

## Layout

- Page margins: 1 inch all sides (standard)
- Line spacing: 1.15
- Paragraph spacing: 6pt after
- Section spacing: 18pt before each new section heading

## Tables

- All borders visible, black, 1pt
- Header row: bold text, no fill
- Cell padding: 4pt
- Align text left in all cells

## Page target

2-3 pages total. If the report exceeds 3 pages, cut Section 2 first (tighten
insights), then Section 5 (trim next-week detail).

## Punctuation

No em dashes anywhere in the document. Use commas, periods, colons, or
semicolons instead. If a sentence needs an em dash, rewrite it as two sentences.

## Output

Save to: `reports/WeeklyReport_Week[N]_[YYYY-MM-DD].docx`

Verify: file size > 30 KB, opens without errors.
