# Critical Thinking Prompts — 读任何东西之前先看这个

> **规则**：你读完任何一篇论文、任何一段文档，都要能填下面的表。
> 填不出来 = 你没真正读懂，只是"感觉懂了"。
> AI 生成的文档也一样 — AI 写的东西可能是错的，你必须自己判断。

---

## 每篇论文必填表（5 层拆解）

读完一篇 paper 后，在 `papers/reading-notes/` 对应的笔记末尾追加这个表：

```markdown
## 我的 5 层拆解

### Layer 1: 它到底在回答什么问题？
（不是"这篇 paper 做了什么"。是：它想解决的 scientific question 是什么？）


### Layer 2: 它的核心 claim 是什么？
（一句话：它声称自己做到了什么？）


### Layer 3: 这个 claim 靠什么支撑？
（benchmark? case study? ablation? 可视化? 理论分析?）


### Layer 4: 它偷偷依赖了哪些假设？
（这是 critical thinking 的核心入口。列出至少 2 个。）

1.
2.

### Layer 5: 它最可能错在哪？
（不是为了否定。是为了定位边界。）

```

## 每次 AI 给你解释时，问自己三个问题

AI（包括我 Claude）经常给你一个"看起来完整的解释"。**不要直接接受。**

1. **这个解释是事实，还是 AI 的推断？**
   - 比如我说"charge = -2"，你要问：这个 -2 是哪篇论文实验测的？还是 AI 推导的？
   - 答案：Caulkins 2014 用 NMR 实验测的。这就是事实。

2. **如果换个条件，还成立吗？**
   - 比如 charge = -2 是在 Ain 中间体的情况。换成 Aex1（外部亚胺），charge 还是 -2 吗？
   - 答案：不一定。Aex1 多了 serine，电荷分布会变。

3. **这里有没有被跳过的步骤？**
   - 比如我说"用 antechamber 就可以得到力场参数"。你要问：antechamber 怎么知道哪些原子是芳香性的？它需要氢原子才能判断（FP-010）。

## 本项目中你应该能回答的问题（按难度递进）

### Level 1: 定义（你必须能秒答）

- [ ] MetaDynamics 和普通 MD 的区别是什么？
- [ ] Path CV 的 s(R) 和 z(R) 分别代表什么？
- [ ] COMM domain 是什么？为什么它重要？
- [ ] PLP 是什么？为什么它需要"参数化"？
- [ ] ff14SB 和 GAFF 分别管什么？

### Level 2: 假设（开始有 research 味道）

- [ ] Osuna 用 Path CV 而不是 RMSD 做 MetaD，背后的假设是什么？
  - 提示：Path CV 假设 O→C 转变可以用一条线性路径描述。真的吗？
- [ ] 500 ns 常规 MD "不够" → 需要 MetaD。这个判断靠什么？
  - 提示：看 SI Figure S2 的 RMSD 图。
- [ ] RESP 电荷用 HF/6-31G(d) 算。为什么不用更高精度的方法？
  - 提示：RESP 的设计就是配合 HF/6-31G(d) 的，因为 AMBER 力场参数是在这个 level 上拟合的。

### Level 3: 证据（你在做真正的批判性阅读）

- [ ] Figure 2 的 FEL 看起来很漂亮。但它的收敛性怎么验证的？
  - 提示：看 SI Figure S4-S5 的 ΔΔG vs time 图。
- [ ] 原文说 PfTrpB0B2 的 kcat 比 PfTrpS 高。FEL 真的能解释 kcat 差异吗？
  - 提示：FEL 给的是热力学（ΔG），kcat 是动力学。两者什么关系？
- [ ] Peer review 指出原文没有 error bar。这意味着什么？

### Level 4: 边界（接近好的研究感觉）

- [ ] 这个方法在哪些系统上可能失效？
  - 提示：如果 O→C 转变不是沿一条路径，而是有多条平行路径呢？
- [ ] 用 S. typhimurium 的 3CEP 作为 Closed 参考结构合理吗？
  - 提示：不同物种的 TrpS，Closed 态一样吗？
- [ ] 10 个 walker 够吗？怎么判断？

### Level 5: 迁移（未来做 idea 的入口）

- [ ] 这套 MetaD + Path CV 方法能不能用在其他变构酶上？需要改什么？
- [ ] GenSLM 生成的突变体，用 MetaD 筛选的假设是什么？这个假设可靠吗？
- [ ] 如果 AlphaFold 能预测构象变化，还需要 MetaD 吗？

## 防止"过早相信顺滑叙事"检查清单

每次你觉得"我懂了"的时候，过一遍这个 checklist：

- [ ] 我能不看笔记，用自己的话讲出来吗？
- [ ] 我能指出这个说法依赖的 2 个假设吗？
- [ ] 我能说出它最可能错在哪吗？
- [ ] 如果 PI 问我"为什么是这样而不是那样"，我答得上来吗？

4 个都能打勾 = 你真的懂了。否则回去重读。
